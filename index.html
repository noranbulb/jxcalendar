  • 인풋을 두개 사용하여 시작과 끝을 표현
    사용법
    html : 시작 : <input type="text" id="from-1" name="from-1" data-to="to-1" data-table="table-1"> - 종료 : <input type="text" id="to-1" name="to-1" data-to="from-1" data-table="table-1">
    시작과 종료 input은 data-to에 상대의 id와 같은 값(*중요)을 넣고 data-table(*중요)은 같은 테이블 글자를 넣어야 한다.
    script : new JXCALENDAR( $("#from-1") );
    script : new JXCALENDAR( $("#to-1") );
